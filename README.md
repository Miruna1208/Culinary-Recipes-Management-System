# Culinary Recipes Management System

## Overview

This project is a backend web application developed for managing culinary recipes.  
The system allows storing, retrieving and managing recipes together with their ingredients and preparation instructions.

The application is built using **Java and Spring Boot**, following a layered architecture and exposing REST APIs that allow interaction with the recipe database.

## Technologies Used

- Java
- Spring Boot
- Spring Web
- Maven
- REST API
- JPA / Hibernate
- Relational Database

## Architecture

The application follows a typical Spring Boot architecture:

Controller Layer  
Handles HTTP requests and exposes REST endpoints.

Service Layer  
Contains the business logic for managing recipes and related data.

Repository Layer  
Manages database interactions using JPA repositories.

Database  
Stores recipes, ingredients and related information.

## Features

The system provides functionality for managing recipes, including:

- Creating new recipes
- Retrieving recipe information
- Updating existing recipes
- Deleting recipes
- Managing recipe ingredients
- Structured storage of culinary data
